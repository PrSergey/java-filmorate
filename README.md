## Примеры запросов
### Получение всех пользоватей:
SELECT *
FROM user
### Получение пользователя по id:
SELECT *
FROM user
WHERE user_id=5
### Получение пользователя по id:
SELECT *
FROM user
WHERE user_id=5
### Получение друзей пользователя:
SELECT user_from_id
FROM user
WHERE user_id=5
